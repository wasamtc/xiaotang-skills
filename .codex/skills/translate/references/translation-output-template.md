# Translation Output Template

Use this template as the default output layout.

```markdown
原作者：<作者名或未知>
原文链接：<原始链接或未知>
说明：本文由 AI 翻译，尽量保持原意与原始结构；术语在首次出现时提供中英文对照。

# <译文标题>

![保留原图或原始 HTML 图片块]

<按原文结构翻译后的正文>
```

## Notes

- Keep the metadata block at the beginning of the article.
- If the source has multiple authors, list them in source order.
- If the source has a subtitle, place it under the translated title.
- If the source begins with a hero image, keep it below the title unless the user asks for another position.
- Keep section headings aligned with the original hierarchy.
